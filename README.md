# Counter App - Debugging Task

## Bugs Identified

1. CSS Syntax Errors:
   - Missing semicolon after align-items: center
   - Missing semicolon after border property

2. JavaScript ID Error:
   - Used wrong ID "counter"
   - Correct ID is "count"

3. Missing semicolon in JS

## Fixes Applied

- Added missing semicolons in CSS.
- Corrected getElementById from "counter" to "count".
- Ensured consistent use of innerText.

## Result

The counter app now:
- Increases count on "+"
- Decreases count on "-"
- Properly updates UI without errors.
